# Generation prompts — copy-paste ready

Workflow: generate the **backdrop image first**, then feed that image into an
**image-to-video** model with each clip prompt. Never text-to-video directly —
i2v is what keeps the framing identical so the in-app cut from still to clip
is invisible.

Every prompt below is complete and standalone.

## After generating

```sh
# crop/scale to 1080x1920, cap at 4s, strip audio, HEVC compress
for f in raw/*.mp4; do
  ffmpeg -i "$f" -t 4 -an \
    -vf "scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920" \
    -c:v libx265 -crf 28 -tag:v hvc1 "out/$(basename "$f")"
done
```

Save as `<circuitID>_paddock_flyby1..5.mp4`, backdrops as
`<circuitID>_paddock_backdrop` in the asset catalog. Free circuits first:
`monteCarlo`, `midlands`, clips 1–2.
